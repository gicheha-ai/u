# EUR/USD Forex Predictor

A web application that predicts EUR/USD forex movements and identifies optimal entry points using machine learning.

## Features
- Real-time EUR/USD price data from Alpha Vantage
- Machine learning predictions using Random Forest classifier
- Entry point detection with confidence scores
- Interactive chart with buy/sell markers
- Automatic API key rotation
- Responsive design
